## General Information

### Kit Characteristics

All kaltokri DIY kits share these features:

- **Solder-free assembly:** Pre-crimped wiring eliminates soldering requirement
- **PETG 3D-printed components:** Rigid mechanical parts with necessary hardware included
- **VPforce electronics:** Uses official VPforce motor kits and USB boards
- **Evening assembly time:** Most kits can be assembled in one evening with basic tools
- **Community designed:** Based on open-source designs from VPforce Discord community members

### What's Included in All Kits

- PETG-printed mechanical components
- All necessary screws, nuts, washers, pulleys, bearings, and belts
- Pre-crimped power wiring
- Assembly hardware specific to the conversion

### What's NOT Included

- Original pedals or joystick base
- VPforce motor kit (order separately from VPforce)
- Power supply unit (PSU)
- USB cables
- Tools (hex keys, screwdrivers, drills if required)
